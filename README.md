## General info

Trainee-project with server architecture. Server-side is intended to authorize and store some user data.

## Technologies

Server-side of application is presented in two variants of implementation created with:

- Nestjs: 8.0.0
- Typeorm: 0.2.41
- Pg: 8.7.1

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
